Find vector for the collision

\[\vec{C} = p_{2x} - p_{1x},\; p_{2y} - p_{1y}\]

The distance is the magnitude of that vector

\[d = \sqrt{C_{x}\,^2 + C_{y}\,^2}\]

Use the distance to normalize the collison vector

\[\hat{C} = \frac{C_{x}}{d},\; \frac{C_{y}}{d}\]

This normalized vector is the direction of of the collision

Find the relative velocity of p1 to p2

\[\vec{V} = v_{1x} - v_{2x},\; v_{1y} - v_{2y}\]

The speed of the collision is the dot product of the relative velocity and collision direction vectors

\[ speed = \vec{V} \cdot \vec{C} \]

If this speed is negative then objects are moving away from each other. This means their velocity vectors don't need to be updated

If speed is positive then the new velocitys are updated as so

\[ \vec{V_{f}}= \vec{V} \cdot speed \]

\[ \vec{v1} = \vec{v1} - \vec{V_{f}}\]

\[ \vec{v2} = \vec{v2} + \vec{V_{f}}\]
